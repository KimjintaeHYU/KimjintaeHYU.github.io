Module 01
인간 유전체의 구조
인간 유전체는 약 30억 쌍의 염기로 이루어진 이중 나선 DNA로, 23쌍의 염색체에 나뉘어 세포핵 안에 담겨 있습니다. 이 DNA를 한 줄로 펼치면 약 2미터 길이지만, 히스톤 단백질에 감겨 압축되어 핵 속에 들어갑니다.
그림 1. 유전체 구조의 계층 — 세포핵 → 염색체 → DNA 이중나선 → 유전자
🔵 엑솜 (Exome)
단백질을 코딩하는 엑손 부분으로, 전체 유전체의 약 1.5%를 차지합니다. 질병 관련 변이의 약 85%가 여기서 발생하여 임상적으로 가장 중요한 영역입니다.
🟡 인트론 & 비코딩 DNA
전체 유전체의 약 98.5%를 차지하며, 유전자 발현 조절, RNA 스플라이싱, 염색체 구조 유지 등 다양한 기능을 합니다. GWAS 연구에서 중요성이 증가하고 있습니다.
🟣 변이의 종류
SNV(단일 염기 변이), 소규모 삽입/결실(indel), 구조적 변이(SV), 복제수 변이(CNV) 등 다양한 형태로 발생합니다. 이들을 정확히 탐지하는 것이 NGS의 핵심 목표입니다.
Module 02
차세대 염기서열분석 (NGS) 원리
NGS는 수백만~수십억 개의 짧은 DNA 조각을 동시에 읽어내는 기술입니다. 기존 생거 시퀀싱(1세대)의 한계를 극복하여 전체 유전체를 빠르게, 저렴하게 분석할 수 있게 되었습니다.

📚 WGS vs WES vs 타깃 패널
전장 유전체(WGS): 30억 염기쌍 전체 분석, 모든 변이 탐지 가능
전장 엑솜(WES): 코딩 영역 1.5%만 분석, 비용 절감
타깃 패널: 수십~수백 유전자만 선택적 분석, 임상 진단에 최적화
📊 시퀀싱 깊이 (Depth/Coverage)
각 유전체 위치가 평균 몇 번 읽혔는지를 나타냅니다.
30x WGS: 연구용 기준
100x WES: 임상 진단 기준
500–1000x: 액체 생검, 미세 변이 탐지
Module 03
NGS 분석 파이프라인 실습
시퀀서에서 나온 FASTQ 파일이 최종 변이 목록(VCF)이 되기까지의 전 과정을 단계별로 살펴봅니다. 아래 예시는 실제 임상에서 사용하는 GATK Best Practice 워크플로우를 기반으로 합니다.
그림 4. NGS 분석 표준 파이프라인 (GATK Best Practices)
1
FASTQ 파일 — 원시 시퀀싱 데이터
시퀀서에서 출력된 FASTQ 파일은 4줄이 하나의 리드를 구성합니다. 각 염기에는 품질 점수(Phred score)가 부여됩니다.
@HG002_R1_1 SRR11140744.1 length=150리드 ID TAGCTATGCATGCTAGTCATGATCGATAGC...DNA 서열 (150bp) +구분자 IIHHGFFEEEDCBAIII???>===IHG@BCCIIIFFF...품질 점수 (ASCII)
Phred 품질 점수 해석
Q30 = 염기 오류 확률 0.1% (정확도 99.9%). Illumina 리드에서 Q30 이상의 비율이 80% 이상이면 고품질 데이터로 간주합니다.
Q20 = 99% 정확도  |  Q30 = 99.9%  |  Q40 = 99.99%
2
정렬 (Alignment) — BWA-MEM2로 참조 유전체에 매핑
각 리드를 참조 유전체(hg38)에 정렬하여 BAM 파일을 생성합니다.
01_alignment.sh 
# 참조 유전체 인덱싱 (최초 1회)
bwa-mem2 index hg38.fa

# 페어드엔드 리드 정렬 → SAM → BAM 변환
bwa-mem2 mem \
  -t 16 \
  -R "@RG\tID:sample1\tSM:HG002\tPL:ILLUMINA\tLB:lib1" \
  hg38.fa \
  sample1_R1.fastq.gz \
  sample1_R2.fastq.gz | \
samtools sort -@ 8 -o sample1.sorted.bam

# BAM 인덱싱
samtools index sample1.sorted.bam

# 정렬 통계 확인
samtools flagstat sample1.sorted.bam
# 출력 예:
# 156,234,512 + 0 in total (QC-passed)
# 155,891,200 + 0 mapped (99.78% : N/A)
그림 5. 리드 정렬 모식도 — 참조 유전체에 대한 리드 매핑
3
변이 탐지 — GATK HaplotypeCaller
BAM 파일에서 SNV, Indel 등 변이를 탐지하고 VCF 형식으로 출력합니다.
02_variant_calling.sh 
# 중복 서열 마킹
gatk MarkDuplicates \
  -I sample1.sorted.bam \
  -O sample1.markdup.bam \
  -M sample1.metrics.txt

# 염기 품질 점수 재보정 (BQSR)
gatk BaseRecalibrator \
  -I sample1.markdup.bam \
  -R hg38.fa \
  --known-sites dbsnp_146.hg38.vcf.gz \
  -O recal.table

gatk ApplyBQSR \
  -I sample1.markdup.bam \
  --bqsr-recal-file recal.table \
  -O sample1.recal.bam

# 변이 탐지 (HaplotypeCaller)
gatk HaplotypeCaller \
  -R hg38.fa \
  -I sample1.recal.bam \
  -O sample1.g.vcf.gz \
  -ERC GVCF \
  --dbsnp dbsnp_146.hg38.vcf.gz

# ✓ 완료: sample1.g.vcf.gz 생성

Module 04
임상 NGS 적용 사례
NGS 기술은 감염병 진단, 암 유전체 분석, 희귀 유전 질환 진단 등 다양한 임상 분야에서 활용됩니다. 아래 두 가지 대표적인 사례를 통해 실제 임상 워크플로우를 살펴봅니다.
사례 01 — 감염병 유전체 진단
중증열성혈소판감소증후군 바이러스 (SFTSV) 바이러스혈증 진단
SFTSV(Severe Fever with Thrombocytopenia Syndrome Virus)는 부냐바이러스목 페뇨바이러스과에 속하는 RNA 바이러스로, 진드기(주로 작은소피참진드기)를 매개로 전파됩니다. 치사율이 6–30%에 달하는 고위험 감염병으로, 조기 진단이 예후에 결정적입니다.
그림 6. SFTSV 유전체 구조 및 mNGS 진단 워크플로우
mNGS의 SFTSV 진단 장점
기존 RT-PCR 검사 대비 알려지지 않은 변이 바이러스 탐지 가능, 바이러스 전체 게놈 정보 획득, 바이러스혈증 수준(copies/mL)의 정밀 정량화, 신속한 계통분류로 역학 조사 지원이 가능합니다.
임상적 의의
바이러스혈증(viremia)의 수준은 예후와 직결됩니다. mNGS로 측정한 바이럴 로드가 높을수록 중증화 위험이 높습니다. 또한 L 분절 분석으로 바이러스 약제 내성 돌연변이도 모니터링 가능합니다.
sftsv_mngs.sh — SFTSV mNGS 분석 예시 
# 호스트(인간) 리드 제거 → 병원체 리드만 추출
bowtie2 -x hg38_index -1 R1.fq -2 R2.fq \
  --un-conc-gz pathogen_reads.fastq.gz \
  -p 16 -S /dev/null

# SFTSV 참조 게놈에 정렬 (NCBI: NC_018136, NC_018137, NC_018138)
bwa-mem2 mem sftsv_ref.fa \
  pathogen_reads_1.fastq.gz \
  pathogen_reads_2.fastq.gz | \
samtools sort -o sftsv.sorted.bam
samtools index sftsv.sorted.bam

# 커버리지 및 바이럴로드 계산
samtools coverage sftsv.sorted.bam
# 결과 예:
# #rname      startpos endpos  numreads covbases  coverage meandepth

# 합의 서열 생성 및 계통분석

사례 02 — 암 유전체 진단
종양 유전체 분석 (Tumor Genomic Profiling)
암은 유전자 변이의 축적으로 발생하는 유전체 질환입니다. NGS를 이용한 종양 유전체 분석(Tumor Profiling)은 치료 표적 발굴, 예후 예측, 암 진화 추적에 활용되며, 정밀 종양학(Precision Oncology)의 핵심 기술입니다.

동반진단(CDx)이란?
특정 치료제와 함께 FDA/식약처 승인을 받은 진단 검사입니다. 예: Foundation One CDx는 100개 이상의 적응증에 대해 치료 가이드를 제공합니다. 국내에서는 C-CAT(국가암빅데이터허브)를 통해 패널 검사 지원이 이루어집니다.
정밀 종양학의 미래
단일 유전자 바이오마커를 넘어, 멀티오믹스(WGS + 전사체 + 단백체 + 에피지놈) 통합 분석과 AI 기반 예후 예측이 차세대 임상 표준으로 자리잡고 있습니다. 특히 종양 미세환경 분석과 공간 전사체학이 주목받고 있습니다.
