01
유전체와 유전자 — 일반생물학 개론
DNA의 구조, 유전체·염색체·유전자의 관계, 전사와 번역까지. NGS를 이해하기 위해 필요한 최소한의 분자생물학을 빠르게 복습합니다.
→
02
Sanger부터 NGS까지 — 시퀀싱 원리와 분석
Sanger 시퀀싱의 원리, Illumina·ONT·PacBio 3대 NGS 플랫폼 비교, 그리고 FASTQ → SAM/BAM으로 이어지는 분석 파이프라인을 직접 조작해보는 인터랙티브 실습.
→
03
SFTS 바이러스 시퀀싱 실습 — 중개의학적 의미
중증열성혈소판감소증후군(SFTS) 바이러스의 생활사와 분절 유전체 구조, 그리고 viremia 환자 샘플을 시퀀싱해야 하는 임상적·공공보건적 이유를 다룹니다.
→

how to use

이 수업은 이렇게 설계되었습니다

01 읽기

각 모듈은 핵심 개념을 짧은 문단과 다이어그램으로 설명합니다. 암기보다 "왜 이 기술이 필요했는가"에 초점을 둡니다.

02 조작하기

Module 2의 미니 콘솔에서 가상의 FASTQ read를 직접 정렬하고, quality score를 색으로 확인하며 SAM/BAM의 의미를 체득합니다.

03 적용하기

Module 3에서 실제 임상 시나리오(SFTS 환자 viremia 샘플)에 그 개념을 적용해, "왜 시퀀싱이 진단에 필요한가"라는 질문에 답합니다.

for instructors

강의자를 위한 안내

전체 화면 공유나 학생 개별 노트북 접속 모두 가능합니다. 인터넷 연결 없이도 핵심 콘텐츠는 작동하도록 외부 의존성을 최소화했습니다.

⏱ 30분 진행 제안

  • 0–10분 — Module 1 빠르게 복습 (이미 배운 학생이 많다면 5분으로 축소)
  • 10–20분 — Module 2 원리 설명 + 인터랙티브 콘솔 직접 실습
  • 20–30분 — Module 3 SFTS 사례로 "왜"에 대한 답 제시, Q&A
